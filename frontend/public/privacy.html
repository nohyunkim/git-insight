개인정보처리방침

Git Insight는 서비스 운영에 필요한 최소한의 정보만 처리합니다.

수집하는 정보

Git Insight는 사용자가 입력한 GitHub 아이디를 기준으로 공개 API 조회를 수행합니다. 별도의 회원가입 정보나 민감 개인정보를 직접 수집하지 않습니다.

정보 이용 목적

입력된 GitHub 아이디는 활동 분석 결과를 생성하기 위한 목적으로만 사용됩니다.

제3자 제공

서비스 동작을 위해 GitHub API 및 배포 인프라 제공자(예: Cloudflare, Render)와 통신이 발생할 수 있습니다.
