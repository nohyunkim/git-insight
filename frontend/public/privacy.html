개인정보처리방침

Git Insight는 공개 GitHub 활동을 기반으로 분석 결과를 보여주는 웹 서비스입니다. 이 문서는 서비스 이용 과정에서 어떤 정보가 처리되는지, 어떤 외부 서비스와 통신이 이뤄지는지, 광고 및 쿠키 관련 고지가 무엇인지 설명합니다.

핵심 요약

Git Insight는 공개 GitHub 데이터를 분석하며, 로그인 기능 제공을 위해 Google 계정 기반 인증 정보, 프로필 정보, 사용자가 직접 저장한 결과 기록을 함께 처리할 수 있습니다. 민감정보를 별도로 요구하지는 않지만, 서비스 운영과 보안, 오류 대응에 필요한 범위의 기술 로그는 처리될 수 있습니다.

1. 수집 또는 처리하는 정보

서비스는 아래 범위의 정보를 처리할 수 있습니다.

  • Google 로그인 시 Supabase Auth를 통해 처리되는 사용자 식별자, 이메일, 인증 세션 정보
  • 로그인 후 프로필 구성에 사용하는 닉네임, 아바타 이미지 주소 등 프로필 정보
  • 사용자가 검색창에 입력한 GitHub 아이디
  • GitHub 공개 API를 통해 조회한 공개 프로필, 공개 저장소, 공개 이벤트 정보
  • 사용자가 직접 저장한 분석 결과 스냅샷, 저장 날짜, 선택한 기간 정보
  • 오류 대응과 보안 점검을 위한 접속 로그, 브라우저 정보, 요청 시간 등 기본 기술 로그
  • 문의 또는 오류 제보 시 사용자가 자발적으로 제출한 이메일, 설명 내용, 스크린샷 등의 정보

2. 처리 목적

  • GitHub 공개 활동 분석 결과 생성 및 화면 표시
  • Google 로그인 세션 유지, 마이페이지 접근 제어, 사용자 프로필 표시
  • 사용자가 저장한 분석 결과의 재열람, 날짜별 기록 관리, 비교 기반 기능 제공
  • 서비스 품질 개선, 오류 진단, 악성 요청 대응
  • 문의 또는 오류 제보에 대한 회신과 후속 대응
  • 서비스 운영 현황 파악과 정책 준수 확인

3. 보관 기간

분석 요청 과정에서 생성되는 임시 응답 데이터나 캐시 데이터는 서비스 응답과 성능 최적화를 위해 제한된 기간 동안만 유지될 수 있습니다. 다만 사용자가 로그인 후 직접 저장한 분석 결과와 프로필 정보는 사용자가 삭제하거나 계정을 더 이상 사용하지 않을 때까지 보관될 수 있습니다. 운영 로그와 문의 기록은 보안, 장애 대응, 법적 의무 이행에 필요한 범위에서만 보관하며, 목적이 달성되면 지체 없이 파기하거나 더 이상 특정 개인을 식별할 수 없는 형태로 관리합니다.

4. 외부 서비스 및 제3자 제공

Git Insight는 서비스 기능 제공을 위해 GitHub API, Google 로그인, Supabase 인증/데이터 저장, 배포 인프라, 로그/분석 도구, 문의 수집 도구와 통신할 수 있습니다. 이러한 통신은 서비스 제공에 필요한 범위에서만 이뤄지며, 각 서비스의 자체 정책이 함께 적용될 수 있습니다.

  • GitHub API: 공개 프로필, 공개 저장소, 공개 이벤트 데이터 조회
  • Google / Supabase Auth: 로그인 및 세션 관리
  • Supabase Database: 프로필 정보와 사용자가 저장한 결과 기록 보관
  • 호스팅/배포 인프라: 정적 페이지 및 API 제공
  • 문의 수집 도구: 사용자가 직접 제출한 제보나 문의 접수

5. 광고, 쿠키, AdSense 관련 고지

Git Insight가 Google AdSense 또는 유사한 광고 서비스를 사용하는 경우, 제3자 광고 사업자는 쿠키 또는 유사 기술을 사용하여 사용자의 이전 방문 기록에 기반한 광고를 제공할 수 있습니다.

Google은 광고 쿠키를 사용하여 사용자의 방문 기록에 기반한 맞춤형 광고를 제공할 수 있습니다. 사용자는 광고 설정을 통해 맞춤형 광고를 관리하거나, 관련 브라우저 설정을 통해 쿠키를 제한할 수 있습니다.

  • Google 광고 설정: https://adssettings.google.com/
  • YourAdChoices: https://youradchoices.com/

실제 광고 도입 시점에 따라 본 방침은 개인화 광고, 비개인화 광고, 동의 수집 정책에 맞춰 추가로 갱신될 수 있습니다.

6. 이용자의 권리

이용자는 개인정보 처리와 관련한 문의, 정정, 삭제 요청을 할 수 있습니다. 로그인 사용자는 서비스 내에서 저장한 분석 기록을 직접 삭제할 수 있으며, 공개 GitHub 데이터 자체는 GitHub 플랫폼에서 관리되므로 공개 범위 변경이나 원본 데이터 수정은 GitHub 계정 설정에서 직접 진행해야 합니다.

7. 문의

개인정보 처리 또는 광고 쿠키 관련 문의는 서비스 하단의 문의/오류 제보 경로를 통해 접수할 수 있습니다. 문의 접수 시 필요한 범위에서만 추가 정보를 요청할 수 있습니다.

8. 시행일 및 변경

본 방침은 2026년 4월 8일부터 적용합니다.

서비스 기능, 광고 도입 여부, 법령 변경에 따라 내용이 추가 또는 수정될 수 있습니다.
