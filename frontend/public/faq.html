자주 묻는 질문(FAQ)

왜 private 저장소 활동이 적게 보이나요?

Git Insight는 GitHub 공개 API 기반 분석 도구입니다. private 저장소의 세부 활동은 공개 범위 제한으로 인해 결과에 반영되지 않거나 축소되어 보일 수 있습니다.

기간을 바꾸면 결과가 왜 크게 달라지나요?

7일, 30일, 90일, 6개월, 1년은 서로 다른 집계 창(window)입니다. 기간이 길어질수록 누적 이벤트가 증가하고, 활동 패턴도 달라 보일 수 있습니다.

같은 계정인데 잠시 후 숫자가 달라질 수 있나요?

가능합니다. GitHub API 응답 타이밍, 캐시 만료 시점, 공개 이벤트 반영 시차에 따라 수치가 일부 변동될 수 있습니다.

결과 공유는 어떻게 하나요?

결과 화면의 공유/저장 메뉴에서 링크 복사, PNG 저장, PDF 저장을 사용할 수 있습니다. 링크에는 조회한 아이디와 기간 정보가 포함됩니다.

오류나 개선 요청은 어디로 보내면 되나요?

메인/결과 화면 하단의 문의/오류 제보 링크를 통해 접수할 수 있습니다. 재현 정보(아이디, 기간, 캡처)를 함께 보내주시면 확인이 빨라집니다.
